# changelog-gen

Generate formatted CHANGELOG entries from git history between two tags.

## Prerequisites

- Python 3.10+
- Git

## Quick Start

```bash
python3 scripts/run.py v1.0.0 v1.1.0
```

See [references/api.md](references/api.md) for complete documentation.
